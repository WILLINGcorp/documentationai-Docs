<Update label="2026-02-16" description="v0.39.19" tags={["New Features","Fixes","Improvements","Deletions"]}>
  ### This latest update of the **ModelBoard** **Platform** introduces various *new features*, additions to *existing features*, fixes *bugs*, and *removals* of overlapping elements or unnecessary UI elements cluttering the app.

  ## New Features

  ✅ **NEW** Project configuration wizard on first-time opening of the project for the project sender.  

  ✅ **NEW** Wizard sets the type of ownership.  

  ✅ **NEW** Wizard asks to add extra collaborators to the project, either by searching for an existing user or by sending an email invite to a collaborator not yet on ModelBoard.  

  ✅ **NEW** Created a whole new release form signature flow accessible directly from the Compliance tab of the Collab workflow. 

  ✅ **NEW** Signed release forms are backed up in an all new dedicated vAulth section.

  ✅ **NEW** Added more data to vAulth account overview.  

  ✅ **NEW** Updates log trail to the Collab project overview tab, allowing all project collaborators to view every edit applied to the project.  

  ✅ **NEW** Backup codes and account security subtab on the Settings page, allowing users to generate 10 backup codes and download them as a PDF.  

  ✅ **NEW** Implemented Email-less, One-click Passwordless Passkey Authentication for new Passkey registrations. 

  ##### ⚠️ *Existing Passkeys must be deleted and Regenerated to allow for Email-less, One-click Passwordless Passkey Authentication.*

  ## Improvements

  �&#xDE84;**&#x20;Added** Refactored model grid for desktop. New horizontal layout with more detail for each model without opening their profile.  

  🪄 **Added** Implemented the capability to remove a collaborator from a Collab project.  

  🪄 **Added** When<u> two passkeys</u> are registered in the account, *plus* backup codes are<u> generated + downloaded</u>, your account automatically switches to passkey-only authentication for enhanced security.

  ##### ⚠️ *Please make sure to have your Email up to date and to carefully save your Backup Codes in a safe location, whether digitally or on print / handwritten note.*



  ##### 🪄 **Added** Fallback login methods when Passkeys are Not available (ex.: for new devices or for recovery purposes) through Email-OTP, or backup codes.

  ## Bug Fixes

  🛠️ **Fixed** New vAulth notification menu.  

  🛠️ **Fixed** Replaced the second authentication factor with passkeys. Passkeys for ModelBoard are now synced with vAulth. Passkeys are stronger than TOTP  (phishing-resistant, and hardware-backed).  

  🛠️ **Fixed** vAulth's tab navigation.  

  ## Deleted

  **🗑️ Deleted** Decluttered vAulth's interface by removing unnecessary widgets.

  \--- 

  ###### *This version may also include more undocumented changes fixing various small bugs and implementing minor security upgrades.*
</Update>
